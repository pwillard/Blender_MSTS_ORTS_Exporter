This is a scenery update of the Scale Rail version of the Full Bucket Line v3.03 By Richard Rasmussen. This will install a new route and will not interfere with any other version of the Full Bucket routes. For this route to work correctly you will need to have Scale rail v1.7 (File ID 87680), ScaleRoads (File ID 87684) and NewRoads v4.0 (File ID 80272) installed. This route is comprised of 4 files SR_FBLine4.01_Main_Route_Files.zip, SR_FBLine4.01_terrtex.zip, SR_FBLine4.01_Textures1.zip & SR_FBLine4.01_Textures2.zip. the install instructions are in the main route files archive.
